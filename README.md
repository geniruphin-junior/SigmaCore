## Moteur Scientifique

Un moteur mathématique orienté objet qui couvre toutes les notions mathématiques en scientifique(trigo,algebre,stat,...) conçu à base de python avec sympy  et numpy pour :ui offrir un puissance haute performance pour les calculs

## Utilisation

Très simple à utiliser,ce moteur est composé de deux fichiers utiles math.py (le code sympy et numpy et la POO) et app.py qui est l'application console ou vopus allez entrerz vois equations algebriques et trigonométriques avec des options comme 1.algebre ,2.trigo,... il vous suffis d'entrer un nombre qui correspond à un des menus pour commencer vos calculs

## Installation

```
pip install -r requirements.txt
python app.py
```
