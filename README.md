# python - SygmaCore

Un moteur mathématique orienté objet qui couvre toutes les notions mathématiques que nous apprenons en scientifique(trigo,algebre,stat,...) conçu à base de python avec sympy et numpy pour lui offrir une puissance haute et une performance pour les calculs

## Technos utilisés

comme prononcé si haut ce moteur est fait à partir de :

- python comme langage
- sympy
- numpy

## Installation

```
pip install -r requirements.txt
python app.py
```

## Utilité

personnelement moi et mon collegue lors de nos révisions et lors de nos devoirs cette app a était utile car elle nous a aidé à verifier si nos reponse ou nos reflexions étaient bonnes ou pas
